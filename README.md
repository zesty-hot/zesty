# Zesty

Zesty is a adult entertainment app built with Next.js, Prisma, and Tailwind CSS.

## Features

1. Escrow payment feature to protect private industry workers from scams
2. ID verification to protect users from fraud
3. Reputation system to protect workers from bad actors
4. Direct police reporting to protect users from crime
5. Live stream for content creators to connect with fans
6. Private messaging for users to connect
7. Dating for users to find partners
8. Search directory to find companions nearby
9. VIP subscription only content feed
10. Events, meetups, and gatherings for users to socialise
11. Job listings for industry workers
